The `@@warning` decorator is used to selectively turn compiler warnings on or off.

### Example

<CodeTab labels={["ReScript", "JS Output"]}>

```res
// Turn off warning 44
@@warning("-44")

// Turn off warnings 44 and 102
@@warning("-44-102")
```
